 HOW IT WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

USER VISITS /hssc-results
    ↓
ENTERS SEARCH CRITERIA
    ↓
CALLS /api/search-results
    ↓
SEARCHES JSON DATABASE
    ↓
RETURNS CANDIDATE WITH MARKS & RANKS
    ↓
DISPLAYS BEAUTIFUL RESULT PAGE


RANKING CALCULATION:
    ↓
FROM PDFs: Extract Roll, Registration, Category, Sub Category, PWD, Marks
    ↓
SORT BY MARKS (highest to lowest)
    ↓
ASSIGN OVERALL RANK (1, 2, 3, ...)
    ↓
FOR EACH CATEGORY:
  - SORT candidates in category by marks
  - ASSIGN CATEGORY RANK (1, 2, 3, ...)
    ↓
SAVE TO JSON
    ↓
READY FOR SEARCHING

 EXTRACTION PROCESS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

When you run: npm run extract-results

The script will:

1. Download all 12 HSSC PDF files from URLs
2. Parse PDF text to extract candidate records
3. Extract fields:
   - Roll Number
   - Registration Number
   - Category
   - Sub Category
   - PWD Status
   - Marks
4. Remove duplicate entries
5. Calculate Overall Rank (based on marks)
6. Calculate Category Rank (based on marks within category)
7. Save to public/hssc-results.json
8. Display summary with top 10 candidates

Expected time: 2-5 minutes

Q: Search not returning results?
A: Make sure you're searching with existing test data:
   - Registration: 1001234567
   - Roll: 10001

Q: Can't find extraction script?
A: Make sure you're in project root, then run:
   npm run extract-results

Q: "npm: command not found"?
A: Install Node.js from nodejs.org

Q: PDF extraction failing?
A: Check if PDFs are accessible in browser first, then:
   1. Verify internet connection
   2. Check if PDF URLs are correct
   3. See PDF_EXTRACTION_GUIDE.md for more help

Q: Search returns empty but shows sample data?
A: The search term doesn't exist in database
   - Use provided sample numbers to test
   - Run extraction to add real data
