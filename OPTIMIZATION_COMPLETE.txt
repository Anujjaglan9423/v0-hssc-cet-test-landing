HOW TO USE EACH OPTIMIZATION
================================================================================

1. BATCH API CALLS
   import { batchRequests, getBatchResult } from '@/lib/api-batch'
   
   const results = await batchRequests([
     { id: 'user', endpoint: '/api/user' },
     { id: 'tests', endpoint: '/api/tests' }
   ])
   const user = getBatchResult(results, 'user')

2. OPTIMIZE IMAGES
   import { OptimizedImage, HeroImage, CardImage } from '@/components/optimized-image'
   
   <HeroImage src="/hero.jpg" alt="Hero" />
   <CardImage src="/card.jpg" alt="Card" />

3. LAZY LOAD COMPONENTS
   import { DynamicAnalyticsChart } from '@/lib/dynamic-imports'
   
   <DynamicAnalyticsChart />  // Only loads when needed

4. MONITOR PERFORMANCE
   import { initializePerformanceMonitoring } from '@/lib/performance-monitor'
   
   useEffect(() => {
     initializePerformanceMonitoring()
   }, [])

5. VIEW METRICS
   Visit: http://localhost:3000/api/metrics
   View specific: http://localhost:3000/api/metrics?type=api-call

TROUBLESHOOTING
================================================================================

If images aren't optimizing:
  → Check next.config.mjs remotePatterns configuration

If batch API fails:
  → Verify endpoints return valid JSON
  → Check auth headers are being forwarded
  → Review /api/metrics for error details

If metrics aren't appearing:
  → Enable NEXT_PUBLIC_DEBUG_METRICS=true
  → Check browser console for errors
  → Verify /api/metrics endpoint is accessible

If lazy-loaded components don't load:
  → Check console for 404 errors
  → Verify component import path
  → Ensure component exists at expected location
